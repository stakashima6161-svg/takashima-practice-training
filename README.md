# 髙島事務所 実務トレーニング

税務・不動産・相続・リスマネを、実務判断で学ぶ職員教育サイトです。

## 目的

このサイトは、職員が実務で迷わないための訓練用サイトです。
単なる税法暗記ではなく、次の力を育てます。

- 資料を見て、確認すべきことに気づく力
- お客様に安全に質問する力
- 自分で断定せず、主任・所長へ上げる力
- WSCへ簡潔に報告する力
- 不動産経営・相続・リスマネをつなげて考える力

## Ver.1 の内容

- 初級編100問
- ランダム出題
- 分野別出題
- 自動採点
- 分野別得点表示
- 間違えた問題の解説表示
- WSC報告用の結果コピー欄

## 分野

1. 巡回監査
2. 法人税
3. 消費税
4. 所得税
5. 相続
6. 不動産経営
7. リスマネ・経営リスク
8. WSC報告・お客様対応

## 運用方針

最初は初級編100問から始めます。
今後、中級編、上級編、ケーススタディ編を追加していきます。

## 注意事項

この教材は職員教育用です。
税務判断・保険募集・個別提案を職員が単独で行うためのものではありません。
判断が必要な事項は、必ず主任・所長・専門担当者へ確認してください。
