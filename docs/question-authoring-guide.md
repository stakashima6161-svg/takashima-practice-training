# 問題作成ガイド

## 基本方針

問題は、知識暗記よりも実務判断を重視する。

## 問題の型

### 1. 最初に確認すること

「この場合、担当者が最初に確認すべきことはどれですか。」

### 2. 安全な返答

「社長から質問された場合、若手職員として適切な返答はどれですか。」

### 3. WSC報告

「この発言を聞いた場合、WSCにどう報告しますか。」

### 4. 上長相談

「この場面で、主任・所長に上げるべき理由はどれですか。」

## 避ける問題

- 単なる暗記だけの問題
- ひっかけが強すぎる問題
- 実務で使わない細かすぎる論点
- 職員が単独判断するように誘導する問題

## リスマネ問題の注意

若手職員に保険商品提案をさせる選択肢を正解にしない。
正解は、原則として「経営上の影響確認」「所内確認」「募集人・所長へ連携」とする。
