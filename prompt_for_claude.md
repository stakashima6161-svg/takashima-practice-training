# Claude Codeへの依頼文

このフォルダをもとに、GitHub Pagesで公開する「髙島事務所 実務トレーニング」を作成してください。

## 作成したいもの

税務・不動産・相続・リスマネを実務判断で学ぶ職員教育サイトです。

## 必須要件

- `index.html` だけで基本動作するようにしてください。
- 外部ライブラリは使わないでください。
- `data/questions.json` の100問を読み込む構成でもよいですが、GitHub Pagesで確実に動くようにしてください。
- 可能であれば、JSON読込失敗時に内蔵サンプルでも動くようにしてください。
- 分野別フィルターを付けてください。
- 出題数を10問、20問、30問、全問から選べるようにしてください。
- ランダム出題機能を付けてください。
- 採点機能を付けてください。
- 正答率、合格判定、分野別得点を表示してください。
- 間違えた問題は、正解と解説を表示してください。
- 最後にWSC報告用の結果コピー欄を表示してください。

## WSC報告用テキスト

採点後、次の形式でコピーできるようにしてください。

```text
受講日：
受講者：
受講レベル：初級
出題数：
得点：
正答率：
苦手分野：
間違えた主な論点：
次回復習する内容：
主任・所長に確認したいこと：
```

## デザイン

- 職員が使いやすい、やわらかい雰囲気
- PCでもスマホでも使える
- 色は落ち着いた青・グレー系
- 問題文を読みやすくする
- 採点結果を見やすくする

## 注意事項

この教材は職員教育用です。
関与先名、個人情報、金額、証券番号などを入力させないでください。
リスマネ分野では、保険募集に見える表現を避けてください。
職員が一人で判断せず、所長・主任・専門担当者へ上げる設計にしてください。
